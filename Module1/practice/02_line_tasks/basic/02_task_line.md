## "Расстояние между точками"

### Задание

Даны координаты (x,y) двух точек на координатной плоскости. Вычислите расстояние между точками.

### Формат входных данных

Даны 4 целые числа, координаты точек

### Формат выходных данных

Вывести расстояние между точками с заданными координатами

### Решение задачи

ax = int(input("координата х точки а:"))
ay = int(input("координата у точки а:"))
bx = int(input("координата х точки b:"))
by = int(input("координата у точки b:"))
d = ((ax-bx)**2+(ay-by)**2)**0.5
print("расстояние между точками: ",d)
